Secured by oauth_2_0

Headers

  • Authorization: required (string)

    Used to send a valid OAuth 2 access token. Do not use with the "access_token" query string parameter.

HTTP status code 401

Bad or expired token. This can happen if the user or Hancock revoked or expired an access token. To fix, re-authenticate the user.

HTTP status code 403

Bad OAuth request (wrong consumer key, bad nonce, expired timestamp...). Unfortunately, re-authenticating the user won't help here.
